# ecommerceWebTest
ecommerce Web Test with testCafe

1. pre condition to kick start this project
install NodeJS

2. Clone this repo
3. run this command to get dependecies
npm install
then make sure the node module folder created in this root folder

4. Run full suite
npm run E2ETest

5. Run login test
npm run loginTest

6. Run product test
npm run productTest

7. Run logout test
npm run logoutTest

8. Run about test
npm run aboutTest

8. Run reset test
npm run resetTest
